# Reports

Generated by `scripts/run_inventory.py`. Not committed by default (see `.gitignore`).

## Generate on your Mac

```bash
cd /Users/mohdasti/Documents/GitHub/dynamometer-analysis
cp config/paths.example.yaml config/paths.local.yaml   # if needed
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python scripts/run_inventory.py
```

Outputs:

| File | Description |
|------|-------------|
| `dataset_inventory.md` | Human-readable summary (paths, tasks, durations, force ranges, Nov2025 linkage notes) |
| `gripforce_file_inventory.csv` | One row per `*_gripforce.csv` with parsed metadata and per-file stats |

After a successful local run, you can commit the generated reports if you want them versioned in git.
